# Portfolio Development Rules
* Use Next.js (App Router), TypeScript, and Tailwind CSS.
* Animations must use Framer Motion, keeping durations under 0.5s and `y` displacement under 30px.
* Respect user accessibility preferences by including Tailwind's `motion-reduce:` modifiers or Framer Motion's `useReducedMotion`.
* Ensure a polished, professional, and academic layout suitable for recruiter evaluation.
* The design must follow a clean, high-whitespace minimalist single-page scroller with a sticky navigation sidebar or top bar.
